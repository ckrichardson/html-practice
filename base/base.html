Base

The <base> element is used to specify the base URL for all relative URLs

To refresh your memory, here are a couple examples of absolute URLs

https://www.google.com

https://www.github.com

Here are a couple examples of relative URLs

./picture.png

./content/cat

An important note about the <base> element is that it must be included in the <head> element of the HTML document

This document has the base set to stackoverflow.com
